# Ghostty 设置

Ghostty的配置会放在一个`/ghostty`目录下。
默认的路径很深，比较合适的是放在`~/.config/ghostty`目录下。

## 配置文件
[hacker_config](/hacker_config)是我的配置文件。

(注意，你的目录中，配置文件名就应该叫`config`。)

安装后，直接把你的ghostty终端变成黑底绿字。
（典型的黑客风格，直接**黑进隔壁班金发小美女的心**）。

- 注意快捷键(`keybind`区)。有一个Quick Terminal很好用，可以快速呼出一个终端。

我这里是control + ~。你可以换成别的快捷键。

- 我这里还提供一个透明风格的ghostty，你可以参考。
[transparent_config](/transparent_config)。

这配合MacOS的“液态玻璃”主题，效果相当好看。

## 分屏操作
在ghostty上，可以用快捷键多开，直接变成多monitor大神。全局爬取视奸所有漂亮妹妹朋友圈。
- cmd+shift+D 上下分屏
- cmd+D 左右分屏
