# 领域展开设置

## 原理

- 用黑底绿色字体展示黑客风格。
- 用分窗来排版展示各个功能区。
- 用键盘输入模拟，实现自动分屏。进一步实现功能区的排版。
- 封装成指令，像魔法一样一键领域展开。

## 工作区设计
我的工作区排版如下：
- 左侧：Crush工作区。Crush就是项目开发的主力。
- 右侧：
  - 顶部：指向pwd的分终端
  - 中部：yazi TUI，追踪项目结构
  - 底部：lazygit展示git仓库状态（AI做了哪些操作）

## 实现方法
以我的工作区排版为例，直接在~/.zshrc中添加[附加指令](.zshrc)

然后source一下即可。

## 完整操作逻辑

1. 打开终端，切换入全屏(对于MacOS，直接cmd+Enter)

2. 按y切除yazi，导航到项目根目录

3. 输入pl（附加指令名），实现领域展开。
